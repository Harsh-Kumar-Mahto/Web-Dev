Display Block

Block elements like div,p,h1..etc have following properties:
1.New line 2.Take whole width of the line 3.Respect custom height and width 4.Respect custom padding and margin

Display Inline

Inline elements like a,span.. have following properties: 1.Try to be in same line 2.Take width according to the content 3.No custom height or width applied 4.Custom padding is available but custom margin only apply in left and right. Inline to block conversion can be done vice versa

Display Inline Block

This is originally an inline element but following block properties can be added to it Custom height and width can be added to it but try to come in same line
