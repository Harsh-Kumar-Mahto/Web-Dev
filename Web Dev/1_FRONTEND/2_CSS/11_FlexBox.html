Flex:Makes a responsive design,content automatically squeze and adjust according to screen size
Flex Wrap:width doesn't squeze unless space is there,they will shift to another line on screen
When flex direction is row main axis is horizontal and when it is column main axis is vertical
Justify content:Place the elements and items according to main axis
Align items:Place the elements and items according to cross axis

Align content:Manages the placement of content & spacing in between multiple lines of the content
