To create a custom react app we follow these stpes:
    1. Create a html file and link it with a js file & this js file is the custom react file.
    2. In the js file, fetch the body of the index file so we can make changes
    3. Create a skeleton of a element which will be followed whenever a new element needs to be created
    4. Create a function that can render the element to the body
    5. Call the render function and create a new element in the body and the function will append the element in the body
