Static Positioning

1.Default positioning
2.Top,Left,Right and Bottom spacing can not be given

Relative Positioning

1.Done w.r.t current window
2.Top,Left,Right and Bottom spacing can be given
3.Space of the element is maintained

Absolute Positioning

1.Done w.r.t closest positioned ancestor/relative
2.Top,Left,Right and Bottom spacing can be given
3.Space of the element (img2) is occupied by the next element

Fixed Positioning

1.Done w.r.t current window
2.Top,Left,Right and Bottom spacing can be given
3.Space of the element is occupied by the next element but on scrolling also the element (img3) remain in its assigned position throughout the page

Sticky Positioning

1.Done w.r.t closest positioned ancestor/relative
2.Top,Left,Right and Bottom spacing can be given
3.Space of the element is maintained and on scrolling,the element remain in it assigned position only inside the closest postioned ancestor
